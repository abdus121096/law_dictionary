# Law Dictionary

Мобильное приложение-справочник по юридическим кодексам.

## Возможности
- Просмотр статей кодексов по главам
- Поиск по ключевым словам
- Избранное

## Стек
- Flutter, Dart
- Локальное хранилище данных (JSON)
